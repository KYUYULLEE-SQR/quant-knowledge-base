# Admin (KB Management)

Knowledge Base 관리, 아키텍처 결정, 템플릿.

## Structure

```
_admin/
├── decisions/        # 중요 아키텍처 결정 기록
├── templates/        # 문서 템플릿
└── pending/          # 반영 대기 중인 업데이트
```

## Purpose

- 중요한 설계 결정을 기록 (왜 이렇게 했는지)
- 새 문서 작성용 템플릿
- KB 업데이트 staging 영역
