## Core Principles

1. **Single Variable**: Phase 1에서 한 번에 하나씩
2. **Baseline Required**: 항상 비교 대상 포함
3. **Falsification**: Signal shift, placebo 테스트
4. **Reproducibility**: 코드/설정/시드 기록
