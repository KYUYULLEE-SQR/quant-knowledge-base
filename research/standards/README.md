연구 표준 및 정책.

| File | Purpose |
|------|---------|
| `backtesting_nav_policy.md` | NAV/MDD 계산 정책 |

## Key Standards

- **NAV**: Mark-to-Market (daily)
- **MDD**: Peak-to-trough, NAV 기준
- **Sharpe**: Annualized, 365-day basis
- **Win Rate**: Trade count 기준 (not P&L weighted)
